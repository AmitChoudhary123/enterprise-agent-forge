# Governance Patterns

Enterprise Agent Forge treats governance as runtime behavior, not a slide at the end.

## Controls included

- Approval required for high-risk work
- Financial controller review for large-dollar workflows
- Block condition for critical high-value actions
- Audit trail for request, plan, tool execution, and policy decision
- Explicit rollback/post-action review controls for high-risk actions

## Why this matters

Agentic AI systems will increasingly perform business operations. The useful question is not only whether an agent can complete a task. It is whether the task was completed with the right controls, evidence, and accountability.
